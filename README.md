# Express-Order

A TypeScript-based order management API built with Express.js and PostgreSQL.

## Overview

Express-Order is a backend application for managing orders with a RESTful API. It provides a foundation for order processing systems with database integration using PostgreSQL through the Neon serverless database.

## Tech Stack

- **Runtime:** Node.js
- **Language:** TypeScript
- **Framework:** Express.js 5.2.1
- **Database:** PostgreSQL (Neon Serverless)
- **Database Driver:** pg
- **Development Tools:** tsx, TypeScript

## Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn
- PostgreSQL database (or Neon serverless account)
